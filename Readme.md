# DSA Preparation --> C++

1. cd desired folder you wanted
```bash
cd <foldername>
```
2. Make empty folder name outs in target folder
```bash
mkdir outs
```
3. compile the file
```bash
g++ file_name.cpp -o outs/file_name.exe
```
4. run the file
```bash
outs/file_name.exe
```

- Correct manner to learn
1. Arrays
2. Hashing / HashMaps
3. Two Pointers / Sliding Window
4. Recursion + Backtracking
5. Linked Lists
6. Stacks & Queues
7. Trees
8. Graphs
9. DP (Dynamic Programming) [Optional]
